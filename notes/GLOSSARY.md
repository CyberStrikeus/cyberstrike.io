# Cyberstrike Glossary / Terimler Sozlugu

## A

### Arsenal
Cyberstrike'in AI Gateway servisi. Eski adi "Zen" (opencode'dan). 
AI isteklerini yonlendirir, faturalandirma yapar, format donusumu saglar.

### ACP (Agent Communication Protocol)
Cyberstrike'in agent'lar arasi iletisim protokolu.

## B

### Bun
Node.js alternatifi JavaScript runtime. Daha hizli, all-in-one (runtime + package manager).

### BYOK (Bring Your Own Key)
Kullanicinin kendi API key'lerini kullanmasi. Arsenal bypass edilir.

## C

### Cloudflare Workers
Serverless function platformu. Arsenal backend burada calisir.

### Cyberstrike
Pentest/guvenlik odakli AI coding assistant. Opencode fork'u.

## D

### Drizzle ORM
TypeScript-first database ORM. TiDB ile kullaniliyor.

## H

### Hydra
Cyberstrike'in ucretsiz AI modeli. Eski adi "Big Pickle" (opencode'dan).
Open source kullanicilar icin API key gerektirmeden erisim saglar.

Mitoloji: Yunan mitolojisinde cok basli canavar. Kesilen her bas icin iki yeni bas cikar.

## O

### Opencode
Cyberstrike'in fork'landigi acik kaynakli proje.

### OpenRouter
Birden fazla AI provider'a tek API ile erisim saglayan servis.

## P

### Provider
AI model saglayicisi (Anthropic, OpenAI, Google, Arsenal, vb.)

## R

### R2
Cloudflare'in object storage servisi (S3 alternatifi).

## S

### SST (Serverless Stack)
Infrastructure as Code framework. AWS/Cloudflare kaynaklarini kod ile yonetir.

### Stripe
Odeme isleme platformu. Arsenal billing icin kullanilir.

## T

### TiDB
MySQL-compatible distributed database. Serverless versiyonu ucretsiz tier sunuyor.

### TUI (Terminal User Interface)
Terminal icinde calisan grafik arayuz. Cyberstrike'in interaktif modu.

## Z

### Zen (Eski)
Arsenal'in eski adi (opencode'dan). Artik kullanilmiyor.
