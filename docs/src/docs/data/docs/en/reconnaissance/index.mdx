Reconnaissance is the first phase of penetration testing. Cyberstrike integrates powerful recon tools.

## Subdomain Enumeration

### Subfinder
Fast passive subdomain discovery:

```bash
cyberstrike> Use subfinder to enumerate subdomains of example.com
```

### Amass
In-depth subdomain enumeration:

```bash
cyberstrike> Run amass for comprehensive subdomain discovery on example.com
```

## DNS Reconnaissance

### DNSx
DNS resolution and analysis:

```bash
cyberstrike> Resolve all discovered subdomains using dnsx
```

## Web Reconnaissance

### HTTPx
HTTP probing and analysis:

```bash
cyberstrike> Probe all subdomains with httpx to find live web servers
```

### Katana
Web crawling and content discovery:

```bash
cyberstrike> Crawl https://example.com with katana
```

## OSINT Tools

### TheHarvester
Email and subdomain harvesting:

```bash
cyberstrike> Gather emails and subdomains for example.com using theHarvester
```

<Aside variant="tip">
  Combine multiple tools for comprehensive reconnaissance. Cyberstrike can chain tools automatically.
</Aside>
