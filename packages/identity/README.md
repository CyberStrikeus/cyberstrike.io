# Cyberstrike Brand Assets

This directory contains the official Cyberstrike brand assets.

## Files

- `mark.svg` - Logo mark (dark background)
- `mark-light.svg` - Logo mark (light background)
- `mark-*.png` - PNG exports at various sizes

## Usage Guidelines

When using these assets:

1. Do not modify the colors or proportions
2. Maintain adequate spacing around the logo
3. Use the appropriate variant for light/dark backgrounds
