# @cyberstrike/function

Serverless functions for Cyberstrike backend services.

## Overview

This package contains standalone serverless functions that run on Cloudflare Workers, handling background tasks and integrations.

## Features

- GitHub App authentication and webhook handling
- JWT token management
- Hono-based HTTP handlers

## Dependencies

- `@octokit/auth-app` - GitHub App authentication
- `@octokit/rest` - GitHub REST API client
- `hono` - Lightweight web framework
- `jose` - JWT operations
