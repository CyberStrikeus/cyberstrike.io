# @cyberstrike/util

Shared utility functions and helpers for Cyberstrike packages.

## Overview

This private package provides common utility functions used across the Cyberstrike monorepo.

## Usage

```typescript
// Import specific utilities
import { someUtil } from "@cyberstrike/util/filename"
```

## Features

- Zod schema helpers
- Common TypeScript utilities
- Shared helper functions

## Development

```bash
cd packages/util
bun run typecheck
```
